# Wild Path Game

Practical survival-training puzzle game built around realistic situations, hidden-object decisions, inventory limits, and short explanations of why each item can help or hurt in a real scenario.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Documentation

The canonical project package is in `docs/`. It defines the product direction, game design, learning model, content rules, technical direction, risks, release criteria, and change-control process.

After the documentation lock, future fundamental changes must go through a change request or ADR.
